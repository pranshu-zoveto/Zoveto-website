## Review Rules

- Ask only real users or customers for reviews.
- Do not incentivize fake or misleading reviews.
- Do not add review or aggregate rating schema to the website until public verified reviews exist.
- Keep claims consistent with current product capabilities and public pricing.

## Launch Sequence

1. Create all profiles with identical entity information.
2. Add screenshots and pricing links.
3. Verify canonical website links point to `https://zoveto.com`.
4. Add directory profile URLs to an internal tracking sheet.
5. Start review requests only after the profile is approved and customer usage is real.
